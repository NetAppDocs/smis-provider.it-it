= dischi smi
:allow-uri-read: 
:icons: font
:imagesdir: ./media/


[role="lead"]
Il `smis disks` comando visualizza le informazioni sul disco per i sistemi di archiviazione. Il `smis disks` comando Ã¨ supportato solo nei controller ONTAP 7-Mode.

.Sintassi
dischi smi

`[-t {*http* | *https*}]`

.Posizione
`C:\Program Files (x86)\NetApp\smis\pegasus\bin`

.Livello di privilegio
Un utente con un nome utente e una password validi

.Parametri
* `*[-t {http | https}]*`
+
Protocollo da utilizzare: HTTPS (impostazione predefinita) o HTTP



.Esempio
Il `smis disks` comando e relativo output abbreviato:

----
smis disks
